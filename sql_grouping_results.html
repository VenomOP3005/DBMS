SQL GROUP BY Statement

The GROUP BY statement groups rows that have the same values into summary rows, like "find the number of customers in each country." It is often used with aggregate functions (COUNT(), MAX(), MIN(), SUM(), AVG()) to perform a calculation on each group.

Syntax

SELECT column_name(s) FROM table_name WHERE condition GROUP BY column_name(s) ORDER BY column_name(s);

Example

The following SQL statement counts the number of customers in each country:

SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country;

The following SQL statement lists the number of customers in each country, sorted high to low:

SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country ORDER BY COUNT(CustomerID) DESC;
