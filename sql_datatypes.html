SQL Data Types for MySQL, SQL Server, and MS Access

Data type Description
CHAR(size) Fixed length string (can contain letters, numbers, and special characters). The size parameter defines the column length in characters - can be from 0 to 255. Default is 1
VARCHAR(size) Variable length string. Max size is 65535 characters
TEXT(size) Holds a string with a maximum length of 65,535 characters
TINYINT(size) Allows whole numbers from -128 to 127 or 0 to 255
SMALLINT(size) Allows whole numbers from -32,768 to 32,767 or 0 to 65,535
MEDIUMINT(size) Allows whole numbers from -8,388,608 to 8,388,607 or 0 to 16,777,215
INT(size) Allows whole numbers from -2,147,483,648 to 2,147,483,647 or 0 to 4,294,967,295
BIGINT(size) Allows whole numbers from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807 or 0 to 18,446,744,073,709,551,615
DECIMAL(p,s) Exact fixed-point number. The total number of digits is specified in p, and the number of digits after the decimal point is specified in s
FLOAT(p) Floating point number. Can hold approximate values
DATE A date. Format: YYYY-MM-DD
DATETIME(fsp) A date and time combination. Format: YYYY-MM-DD HH:MI:SS
TIMESTAMP(fsp) A timestamp. Format: YYYY-MM-DD HH:MI:SS
TIME(fsp) A time. Format: HH:MI:SS
YEAR A year in four-digit format
