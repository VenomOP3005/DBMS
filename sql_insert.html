SQL INSERT INTO Statement

The INSERT INTO statement is used to insert new records in a database table.

Syntax

Inserting into specific columns:

INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);

Inserting into all columns (if values are for every column, in order):

INSERT INTO table_name VALUES (value1, value2, value3, ...);

Example

This example inserts a record into the Customers table:

INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country) VALUES ('Cardinal', 'Tom B. Erichsen', 'Skagen 21', 'Stavanger', '4006', 'Norway');

In this example, the CustomerID column is not included because it is assumed to be auto-incremented.

Output

+------------+--------------+----------------+-----------+-----------+------------+---------+ | CustomerID | CustomerName | ContactName | Address | City | PostalCode | Country | +------------+--------------+----------------+-----------+-----------+------------+---------+ | NULL | Cardinal | Tom B.Erichsen | Skagen 21 | Stavanger | 4006 | Norway | +------------+--------------+----------------+-----------+-----------+------------+---------+

Insert Into Specific Columns

You can also insert data into only selected columns:

INSERT INTO Customers (CustomerName, City, Country) VALUES ('Cardinal', 'Stavanger', 'Norway');

Other columns will use default values or remain NULL if no default is defined.
