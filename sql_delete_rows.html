SQL DELETE Statement

The DELETE statement is used to remove existing records from a table.

Syntax

DELETE FROM table_name WHERE condition;

Note: Be cautious when deleting records. The WHERE clause specifies which record(s) should be deleted. If you omit the WHERE clause, all records in the table will be deleted!

Example

Delete the customer "Alfreds Futterkiste" from the "Customers" table:

DELETE FROM Customers WHERE CustomerName='Alfreds Futterkiste';

After executing the above statement, the "Customers" table will no longer include the customer named "Alfreds Futterkiste".

Delete All Records

It is possible to delete all rows in a table without deleting the table itself. This means that the table structure, attributes, and indexes will remain intact:

DELETE FROM table_name;

For example, to delete all records in the "Customers" table:

DELETE FROM Customers;

Note: Be very careful when deleting records. If you omit the WHERE clause, all records in the table will be deleted!
